## üìã Feature 1: Create Prescription

### Step-by-Step Guide:

**Step 1: Navigate to Prescriptions**
- Click **'Prescriptions'** in the navigation menu
- Or click **'Create Prescription'** button on homepage

**Step 2: Fill Patient Information**
- **Patient Name**: Enter full name
- **Age**: Enter patient age
- **Sex**: Select Male/Female/Other
- **Mobile**: Enter 10-digit mobile number
- **Email**: Enter patient email (for automatic email)
- **Address**: Enter patient address

**Step 3: Select Treatment Type**
- Click treatment type dropdown
- Choose from 15+ options:
  - Root Canal
  - Tooth Extraction
  - Dental Filling
  - Braces/Orthodontics
  - Teeth Cleaning
  - Crown/Bridge
  - Implant
  - Wisdom Tooth Removal
  - And more...

**Step 4: Add Medicines**

**Option A: Using Medicine Codes**
1. Enter medicine code (e.g., M001)
2. Click **'Fetch Medicine'** button
3. Details auto-populate from database

**Option B: Manual Entry**
1. Enter medicine name
2. Enter dosage (e.g., 500mg)
3. Select timing (Morning/Afternoon/Night)
4. Enter duration (e.g., 5 days)
5. Add instructions (before/after food)

**Add Multiple Medicines:**
- Click **'Add Medicine'** button for more rows
- Remove unwanted rows with **'Remove'** button

**Step 5: Treatment Notes**
- Enter detailed treatment notes
- Add special instructions
- Include precautions if any

**Step 6: Set Follow-up Date (Optional)**
- Click date picker
- Select future date
- System sends automatic reminder 1 day before

**Step 7: Upload X-Ray (Optional)**
- Click **'Select X-ray Image'**
- Choose image file (JPG, PNG)
- Preview appears automatically
- Click **'Analyze with AI'** for automatic analysis

**Step 8: Save Prescription**
- Click **'Save Prescription'** button
- Prescription saved to database
- Email sent automatically to patient
- Success message appears

**Step 9: Print Prescription**
- Click **'Print Prescription'** button
- Print preview opens
- Select printer or **'Save as PDF'**
- Professional format with logo

**Chatbot Help:** Ask "How to create a prescription?" or "How to add medicines?"

## üè• Feature 2: X-Ray Upload & AI Analysis

  - Severity level
  - Confidence score
  - Treatment recommendations

**Step 8: Include in Prescription**
- X-ray automatically linked to prescription
- AI results included in patient email
- Visible in prescription history
